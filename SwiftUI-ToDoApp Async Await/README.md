
# TO-DO SwiftUI 

This **To-Do** app was created using `SwiftUI` completely based on `UIKit` version`

**Includes:**
- async / await
- @Observable macro
- UnavaibaleContentView
- Searchable
- View Model
- Text Format
- Swipe actions
- Refreshable
